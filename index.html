Download/Install

Linux

For Linux systems, I will only describe how to install Python on Ubuntu Linux machines, but other distros are very similar and are easy to find with a quick Google Search.

To see which version of Python 3 you may already have installed or confirm that it is not installed, open your command prompt and run:

$ python3 --version

To upgrade to a newer version or install python 3.10 fresh, run the following commands:

$ sudo apt-get update $ sudo apt-get install python3.10

Variables

This section is about how to use variables in Python.

Variable name starts with alpha or _. Variables cannot contain !,@,#,$ and so on within the variable's name.

Variable named (a) and being assigned the integer value of (5)

a = 5

Variable named (b) and being assigned the String value of ("Hello World")

b = "Hello World"

If/elif/else Statement

This section covers how to use if-elif-else statements.

if number > 0: print("Number is positive !!") elif number == 0: print("Number is 0") else: print("Number is Negative !!")

With the code above, see what is outputted when you create a variable named number and set it to 0, then 25, then finally -200.

Loops

This section covers the two different loops available in Python, For-Loop and While-Loop

For Loop

for i in range(5): print(i)

This will loop through the numbers from 1-4, printing each out to the terminal.


While Loop

i = 0 while <= 5: print(i) i = i + 1

This While Loop will print out the same thing as the For Loop above.
