Data Types(Primitive)

This section deals with explaining the different primitive data types that are available in Python.

  • Integer - Whole Numbers - 5, 8, 0, -85
  • String - In Between ""/'' - "Hello"
  • Float - Decimal Number - 2.3 0.0 -9.99
  • Boolean - True or False

Variables

This section is about how to use variables in Python.

Variable name starts with alpha or _. Variables cannot contain !,@,#,$ and so on within the variable's name.

Variable named (a) and being assigned the integer value of (5)

a = 5

Variable named (b) and being assigned the String value of ("Hello World")

b = "Hello World"

Arithmetic Operators

This section covers all of the available Operators to perform Arithmetic in Python.

  • + for Addition > 100 + 150 = 250
  • - for Subtraction > 50 - 30 = 20
  • * for Multiplication > 3 * 3 = 9
  • / for Division > 900 / 10 = 90
  • % for Modulus > 7 % 4 = 3
  • ** for Exponentiation > 4**2 = 16
  • // for Floor Division > 7 // 2 = 3

Comparison Operators

This section covers all of the available Operators to Compare different pieces of data in Python.

  • == is equal to
  • != not equal to
  • < less than
  • > Greater than
  • <= Less than or equal to
  • >= Greater than or equal to
